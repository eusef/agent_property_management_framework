# Unit Turnover Checklist

**When to use:** Between tenants. Start on move-out day. The goal is to minimize vacancy days while bringing the unit to rent-ready condition.
**Key principle:** Start with the longest tasks first (painting, flooring), run tasks in parallel where possible, and deep clean last.

## Phase 3: Repairs and Replacements (Days 2-5)

### Kitchen
- [ ] Appliances cleaned and tested (oven, fridge, dishwasher, range hood)
- [ ] Faucet working properly, no leaks
- [ ] Cabinets: doors aligned, hardware tight
- [ ] Countertops: no damage, caulk intact
- [ ] Garbage disposal working (or removed - disposals are a top source of maintenance calls)

### Bathroom(s)
- [ ] Toilet: working, no leaks, seat condition: [ ] OK [ ] Replace (replace if stained - cheap, high impact)
- [ ] Tub/shower: caulk intact, no mold, drain clear
- [ ] Faucet working, no leaks
- [ ] Exhaust fan working
- [ ] Re-caulk tub/shower if needed

### Throughout Unit
- [ ] All light fixtures working, bulbs replaced
- [ ] All outlets and switches working (test with outlet tester)
- [ ] All doors open/close/latch properly
- [ ] All windows open/close/lock properly
- [ ] Window screens intact
- [ ] All locks working (re-key exterior locks between tenants)
- [ ] Smoke detectors: tested, new batteries installed
- [ ] CO detectors: tested, new batteries installed

### HVAC
- [ ] Furnace filter replaced
- [ ] System tested (heat and AC)
- [ ] Thermostat working

### Water Heater
- [ ] Age: _____ years (replace proactively at 10-12 years)
- [ ] Working properly, no leaks

## Phase 5: Deep Clean (After All Work Complete)

This is always the LAST step. No point cleaning before painting or repairs.

### Kitchen
- [ ] Oven interior and exterior
- [ ] Refrigerator interior, exterior, and coils
- [ ] Dishwasher interior
- [ ] Range hood and filter
- [ ] Countertops and backsplash
- [ ] Cabinets inside and out
- [ ] Sink and fixtures
- [ ] Floor

### Bathroom(s)
- [ ] Toilet (inside bowl, base, behind)
- [ ] Tub/shower (tile, grout, fixtures)
- [ ] Sink and vanity
- [ ] Mirror
- [ ] Floor

### Throughout
- [ ] All hard floors mopped
- [ ] All carpets professionally cleaned (keep receipt for deposit accounting)
- [ ] Windows cleaned (interior)
- [ ] Light fixtures and ceiling fans dusted
- [ ] Baseboards wiped
- [ ] Closet shelves and rods wiped
- [ ] Garage swept
