# PM_FRAMEWORK

**AI-assisted property management for independent landlords.**

Stop juggling spreadsheets, email threads, and sticky notes. PM_FRAMEWORK gives you a structured, file-based system that works with any AI assistant to help you manage rental properties like a pro вЂ” without the cost of a property management company.

## What is this?

A ready-to-use directory structure + web dashboard for managing rental properties. Everything lives in plain-text files (Markdown, CSV) that you own and control. Point your AI tool at it, and it becomes your property management assistant.

- **Track** rent, expenses, maintenance, and compliance
- **Generate** tenant communications from battle-tested templates
- **Monitor** deadlines, lease renewals, and required inspections
- **Stay compliant** with jurisdiction-specific checklists and calendars

## Quick Start

### Option A: Double-click (recommended)

1. **macOS** вЂ” Double-click `START.command`
2. **Windows** вЂ” Double-click `START.bat`
3. **Linux** вЂ” Run `./START.sh`

The launcher handles everything: checks for Python, creates a virtual environment, installs dependencies, and opens the browser. On first launch, a setup wizard walks you through configuring your properties.

### Option B: Command line

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/PM_FRAMEWORK.git
cd PM_FRAMEWORK

# 2. Launch (auto-installs dependencies, opens browser)
# macOS:
./START.command
# Linux:
./START.sh
```

If the server is already running, the launcher just opens your browser to it.

### Option C: Manual setup (advanced)

```bash
cd webapp
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python -m app
# в†’ Opens http://localhost:8000
```

## First-Run Setup Wizard

On first launch, you'll be guided through an 8-step browser-based setup:

1. **Welcome** вЂ” Overview and getting started
2. **Owner Profile** вЂ” Your name, email, and phone
3. **Jurisdiction** вЂ” Select your state for compliance tracking
4. **Add Property** вЂ” Enter your first rental property
5. **Properties List** вЂ” Review, edit, or add more properties
6. **AI Setup** вЂ” Choose your AI assistant tool
7. **Review** вЂ” Confirm everything before setup
8. **Complete** вЂ” Property directories and config files are created

Your progress is saved at each step вЂ” close the browser and come back anytime. All configuration is stored in a local `config.json` file.

## Web Dashboard

The included web dashboard provides:

- **Browser-based setup wizard** вЂ” No terminal required for first-time configuration
- **File browser** with Markdown rendering and side-by-side editing
- **Property cards** with key metrics pulled from property READMEs
- **Task tracking** with interactive checkbox support
- **Full-text search** across all documents
- **Settings page** вЂ” Edit owner info, properties, and jurisdiction anytime
- **Installable as a desktop app** (PWA) вЂ” Works like a native application
- **Offline support** вЂ” Shows a friendly message when the server isn't running

## Works With Any AI Tool

PM_FRAMEWORK is **tool-agnostic**. The `.pm_agent/AGENT_CONTEXT.md` file serves as the universal entry point:

| Tool | How to use |
|------|-----------|
| **Claude Code** | Automatically reads `.claude/CLAUDE.md` в†’ `.pm_agent/AGENT_CONTEXT.md` |
| **ChatGPT / GPT-4** | Paste or upload `.pm_agent/AGENT_CONTEXT.md` as system context |
| **Cursor** | Add `.pm_agent/AGENT_CONTEXT.md` to your project context |
| **Any LLM** | Point it at `.pm_agent/AGENT_CONTEXT.md` for full project awareness |

You select your AI tool during setup. Change it anytime in Settings.

## FAQ

**Q: Do I need to know how to code?**
No. Double-click the START file, complete the browser wizard, and you're ready. Everything is plain Markdown and CSV files.

**Q: Is my data stored in the cloud?**
No. Everything stays on your local machine in plain-text files you control. No accounts, no subscriptions, no cloud dependencies.

**Q: Can I use this for multiple properties?**
Yes. Add as many properties as you need during setup, or add more later from Settings.

**Q: What about state-specific laws?**
The `compliance/` folder has jurisdiction-agnostic templates. The `examples/oregon/` folder shows how to fill them in for a specific state. Customize for yours.

**Q: Can I install it as a desktop app?**
Yes. When using Chrome, you'll see an "Install as app" prompt. The app works in its own window like a native application.

**Q: Can I use this without an AI assistant?**
Absolutely. It's a well-organized file system with a web dashboard that works on its own. The AI integration is a bonus.

**Q: What Python version do I need?**
Python 3.11 or later. The launcher checks this automatically and shows a clear message if you need to install it.
