# Oregon Example Jurisdiction Package

**Purpose:** This folder contains Oregon-specific compliance documents as a worked example of how to customize the framework's jurisdiction-agnostic templates for a specific state.

Use these files as a reference when filling out the templates in `compliance/` for your own state. They show what a completed set of state-specific documents looks like.

---

## Files in This Folder

| File | Description | Framework Template |
|------|-------------|-------------------|
| `landlord-tenant-law.md` | Oregon landlord-tenant law summary (ORS Chapter 90) | `compliance/landlord-tenant-law.md` |
| `eviction-guide.md` | Oregon eviction procedures and notice periods | `compliance/eviction.md` |
| `fair-housing-additions.md` | Oregon-specific fair housing protections (beyond federal) | `compliance/fair-housing.md` |
| `portland-regs.md` | Portland/Multnomah County local regulations | (no template -- city-specific) |
| `tax-guide.md` | Oregon + federal tax compliance for rental income | (general reference) |
| `calendar-example.md` | Compliance calendar filled in for Oregon | `compliance/calendar.md` |

---

## How to Use These Examples

1. **Read the corresponding framework template** in `compliance/` to understand the structure
2. **Review the Oregon example** to see how state-specific content fills in the placeholders
3. **Research your own state's laws** and create equivalent files
4. **Pay special attention to local regulations** -- Portland layers significant additional requirements on top of Oregon state law. Your city may do the same.

---

## Disclaimer

These Oregon examples are summaries for operational reference, not legal advice. Oregon laws change frequently. If you are operating in Oregon, verify all information is current and consult an Oregon-licensed attorney before relying on these documents for legal actions.
