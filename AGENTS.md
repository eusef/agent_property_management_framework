# Agent Instructions

All agent configuration and context files are in `.pm_agent/`.

**Read these files in order:**

1. `.pm_agent/AGENT_CONTEXT.md` — Project overview, file map, workflows, approval thresholds
2. `.pm_agent/AGENT_SETUP.md` — Detailed working instructions, CSV schemas, session hygiene
3. `.pm_agent/AGENTS.md` — Project structure, webapp details, development info
